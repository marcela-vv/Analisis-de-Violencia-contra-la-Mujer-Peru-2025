# Analisis-de-Violencia-contra-la-Mujer-Peru-2025
Análisis de datos y scraping de denuncias del SIDPOL (Mininter) para monitorear la evolución de la violencia contra la mujer en el Perú durante el 2025.


**1. Pregunta de investigación:**
¿Qué departamentos han mostrado el mayor crecimiento porcentual (aceleración) de denuncias por violencia en el último año, y cómo esto ayuda a identificar las zonas con mayor urgencia de intervención comunicacional?
